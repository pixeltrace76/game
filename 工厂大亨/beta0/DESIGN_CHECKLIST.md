# 制作清单落实表

这份清单对应当前 Godot 测试版。目标是先把挂机工厂的可玩闭环跑通，再逐步扩展内容。

## 1. 核心玩法

- [x] 工厂自动产生金币。
- [x] 玩家用金币雇佣工人。
- [x] 玩家升级工人，提高工人效率。
- [x] 玩家升级工厂，提高整体产能。
- [x] 达到金币门槛后解锁收益更高的新工厂。
- [x] 新工厂重新开始招聘和升级，形成下一轮成长。

## 2. 数值逻辑

- [x] 每个工厂有独立基础收益。
- [x] 每个工厂有独立解锁价格。
- [x] 工人雇佣价格随数量递增。
- [x] 工人升级价格随等级递增。
- [x] 工厂升级价格随等级递增。
- [x] 总收益由所有已解锁工厂叠加。
- [x] 数字显示支持万、亿、兆等中文单位。

## 3. 工厂内容

- [x] 起步小作坊
- [x] 木材加工厂
- [x] 食品加工厂
- [x] 钢铁冶炼厂
- [x] 电子制造厂
- [x] 机器人装配厂

## 4. UI 界面

- [x] 当前金币显示。
- [x] 每秒总收益显示。
- [x] 主菜单入口：进入工厂、设置、退出游戏。
- [x] 工厂界面入口：设置、退出工厂。
- [x] 设置界面：窗口大小选择，只调整外层游戏窗口。
- [x] 游戏内 UI 分辨率固定为 1280×720，不允许玩家调整。
- [x] 设置界面：音效音量调节，可实时生效。
- [x] 设置界面：删除存档按钮。
- [x] 删除存档前二次确认。
- [x] 删除存档后情绪化提示。
- [x] 设置项会随存档保存。
- [x] 下一个工厂解锁目标。
- [x] 解锁进度条。
- [x] 工厂列表。
- [x] 工厂详情面板。
- [x] 雇佣按钮。
- [x] 工人升级按钮。
- [x] 工厂升级按钮。
- [x] 解锁按钮。
- [x] 里程碑目标显示。
- [x] 重置 beta 存档按钮。

## 5. 存档与离线收益

- [x] 自动保存金币。
- [x] 自动保存每个工厂的解锁状态。
- [x] 自动保存工人数量、工人等级、工厂等级。
- [x] 记录退出时间。
- [x] 再次进入时结算离线收益。
- [x] 离线收益效率为 50%。
- [x] 离线收益最多结算 8 小时。

## 6. 当前测试版暂未包含

- [ ] 经理系统。
- [ ] 订单系统。
- [ ] 科技树。
- [ ] 转生系统。
- [ ] 新手引导。
- [ ] 正式音效资源和播放反馈。
- [ ] 正式美术资源。
- [ ] 广告或内购系统。

## 7. 建议下一版优先级

1. 加入经理系统，让玩家可以购买自动招聘或自动升级。
2. 加入订单系统，提供短期目标和一次性奖励。
3. 加入新手引导，让玩家第一次进入时知道该先雇佣工人。
4. 加入基础动画，比如金币跳动、按钮反馈、解锁特效。
5. 继续打磨中文 UI 文案、字体和排版细节。
