# 挂机工厂大亨 Beta v0.1.0

这是用于 GitHub Release 的 Beta 发布包。

## 下载

- macOS: `IdleFactoryTycoon_Beta_v0.1.0_macOS.zip`
- Windows x64: `IdleFactoryTycoon_Beta_v0.1.0_windows_x64.zip`

## 主要内容

- 主菜单与游戏内 UI 美术更新。
- 新增中文主菜单标题 Logo。
- 新增不同工厂背景图。
- 新增成就目标系统。
- 新增离线收益、离线效率升级、点击收益升级。
- 新增换地区系统。
- 新增背景音乐与按钮/点击音效。
- 新增中英文语言切换。
- 新增作弊模式快捷键 `G`。

## 说明

- macOS 版本未做 Apple notarization，如系统提示无法打开，需要在系统设置中手动允许。
- 校验文件：`SHA256SUMS.txt`
