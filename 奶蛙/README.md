# 奶蛙微信小程序

这是一个极简版“奶蛙 - 点一下就狂笑抽象魔性”微信小程序。

## 当前玩法

- 原始奶蛙图片居中展示，代码不重绘、不裁切、不改形象
- 待机和倒计时时播放 `assets/idle.gif`，背景色按素材边缘采样统一为 `#fdfdfd`
- 大笑时显示 `assets/laugh.gif`，已用同段视频音频替换 `laugh.mp3`，并添加安全边距避免人物被裁
- 页面正下方是时、分、秒滚动转轮和一个按钮
- 小时范围 0 到 12，分钟和秒范围 0 到 60
- 倒计时结束后播放奶蛙狂笑
- 笑声从原音轨第 0 秒开始，截取前 7.3 秒循环

## 素材

奶蛙原图：

```text
assets/naiwa.jpg
```

音频放到：

```text
assets/sounds
```

首页会读取这个文件名：

```text
assets/sounds/laugh.mp3
```

## 导入

可以用下面任意一种方式打开：

- 用微信开发者工具导入项目，选择本项目的 `奶蛙` 文件夹。
- 如果已经发布为微信小程序，也可以直接在微信小程序里打开。
